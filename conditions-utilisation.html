Conditions d'utilisation

Bienvenue sur Clara Speaker ! En utilisant notre application, vous acceptez les présentes Conditions d'utilisation. Veuillez les lire attentivement.

1. Description du Service

Clara Speaker est une application mobile qui transforme vos e-mails en résumés audio concis, vous permettant de les écouter comme un podcast. Notre objectif est d'augmenter votre productivité, de réduire votre temps d'écran et de prioriser les informations importantes grâce à l'intelligence artificielle.

2. Connexion et Autorisations

Pour fonctionner, Clara Speaker nécessite que vous connectiez votre compte de messagerie (Gmail) via le protocole OAuth 2.0. Vous devrez accorder les permissions nécessaires pour que l'application puisse :

  • Accéder à votre boîte de réception pour détecter les nouveaux e-mails.
  • Détecter la connexion de vos écouteurs Bluetooth (Android 12+).
  • Afficher des notifications (Android 13+).

Nous ne stockons pas vos identifiants de messagerie. Les jetons d'authentification sont gérés de manière sécurisée via des variables d'environnement.

3. Traitement des E-mails et Confidentialité

  • Le contenu de vos e-mails est traité par notre IA (via l'API Gemini) uniquement dans le but de générer des résumés audio.
  • Le traitement des e-mails est éphémère : le contenu de vos e-mails n'est pas stocké par notre backend après la génération du résumé.
  • Les résumés audio générés sont stockés localement sur votre appareil.
  • Nous utilisons le lastHistoryId de l'API Gmail pour ne traiter que les événements survenus depuis votre dernière utilisation, garantissant ainsi une synchronisation sans faille sans traiter l'historique complet de votre boîte mail.

4. Fiabilité et Performance

Nous nous engageons à ce que Clara Speaker soit fiable et performante. Cependant, nous ne pouvons garantir une disponibilité ininterrompue du service ou l'absence totale d'erreurs. Le système de file d'attente locale (Room) garantit qu'aucun résumé n'est perdu si votre appareil n'est pas disponible pour l'écouter.
